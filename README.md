# Sex-is-the-most-attractive-gender
This repository contains the data, code and analyses for the paper "Sex is the most attractive gender: the birth and death of noun categorization in language", Authors: Carling, Gerd, Nour Efrat-Kowalsky, Niklas Erben-Johansson, Gerhard Jäger, Lev Michael, Marc Allassonnière-Tang, and Filip Larsson.
All data used in this paper is found in the csv files available in this repository and are called for in the script as necessary.
The R script generates the data for the phylogenetic analysis that was run on BayesTraits V4. In order to run the phylogenetic analyses, BayesTraits needs to be located in the same folder as the data and config files. The results of our runs are available in this repository and are consequently visualised in the R script that is available here.
